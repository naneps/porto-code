# Porto Code — Documentation

> **Porto Code** is an interactive, VS Code-inspired portfolio application built with React 19 + TypeScript + Vite. It lets visitors explore **Nandang Eka Prasetya's** work experience, projects, skills, and contact information through a familiar IDE interface.
